<p>즉, 배열 전체에서 엄밀히 내림차순인 쌍의 개수와 엄밀히 오름차순인 쌍의 개수가 서로 같다는 뜻이다. <strong>같은 값인 쌍은 위 정의에서 세지 않는다.</strong></p>

<p>도사의 도치균형의 배열을 보며 와우가 물었다.</p>

<p>"모든 배열이 같은 값으로만 구성되면 도치균형의 배열이네요. 너무 시시한데요?"</p>
